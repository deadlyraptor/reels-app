FAQs

Why does this site take so long to load?

The site is hosted on Heroku's free tier. The free web dyn goes to sleep after a 30-minute period of inactivity. If the page is loading slowly, it's because the dyno was sleeping and is taking a couple of minutes to "wake up."

What's with the file naming scheme?

In 2020, I reorganized Coral Gables Art Cinema's entire digital filing system on Google Drive. The file naming convention is meant to make it easy to identify the contents of a file just from its name and also make it easy to search. If you name a Seven Smurai still ss-01, it will not show up in search results if you type "seven samurai."
